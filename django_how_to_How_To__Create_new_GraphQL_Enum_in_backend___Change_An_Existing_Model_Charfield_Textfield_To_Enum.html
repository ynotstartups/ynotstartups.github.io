How To: Create new GraphQL Enum in backend / Change An Existing Model Charfield/Textfield To Enum

  • Create TextChoices from django
  • Create a schema in saltus/oneview/graphql/model_types.py
  • Use this schema in corresponding Mutation

Create TextChoices from django

class OVRecommendActionEnum(TextChoices):
    SET_UP_NEW = "Apply for a new plan"
    WITHDRAW = "Withdraw funds from a plan"
    CONTRIBUTE = "Increment an existing plan"
    TRANSFER = "Transfer to new plan"
    MODIFY = "Modify an existing plan"
    AD_HOC_FEE = "Charge an ad-hoc fee"

Set choices field in corresponding CharField or TextField

    action = models.CharField(
        max_length=255,
        blank=True,
        null=True,
        choices=OVRecommendActionEnum,
    )

Create a schema in saltus/oneview/graphql/model_types.py

OVRecommendActionSchema = graphene.Enum.from_enum(OVRecommendActionEnum)

based on graphene documentation on enum

Use this schema in corresponding model type in saltus/oneview/graphql/model_types.py This is required for graphql query to have Enum type OVRecommendActionEnum

class OVRecommendType(FilteredDjangoObjectType):
    class Meta:
        model = OVRecommend
    ...
    action = graphene.Field(OVRecommendActionSchema)
    ...
    def resolve_action(self, info):
        return self.action

Without manually using OVRecommendActionSchema, make schema will automatically create an enum based on the value of text choice, see this example and this documentation !

By default Graphene-Django will convert any Django fields that have choices defined into a GraphQL enum type. https://docs.graphene-python.org/projects/django/en/latest/queries/

Use this schema in corresponding Mutation

This is required for graphql mutation to accept Enum type OVRecommendActionEnum

class UpsertOVRecommend(graphene.Mutation):
    class Arguments:
        ...
        action = OVRecommendActionSchema(required=True)
        ...

Graphene passes the argument as Enum so remember to use the value when setting the model!

e.g.

            # action has type OVRecommendActionEnum
            kwargs["action"] = kwargs["action"].value
            for key, value in kwargs.items():
                setattr(ovrecommend, key, value)
