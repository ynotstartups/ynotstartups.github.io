What does operation mean?

Lambda abstracts awasy the operational aspects of running code in the cloud, so developers don’t think about selecting instance types, cross-availability zone failover, or patching the operating system, language runtime, or standard and third party libraries. P.S. similar to managing personal laptop, os updates, installing new python, virtualenv…

Operational metrics

  • Latency: client response time, client to server, server to response
  • Throughput: paging rate, CPU utilization, network bandwidth, sustained high measures of anyone indicates an over-subscribed compute host
  • Availability
