Required field

  • -message
  • -list

Useful field

  • -title
  • -sound NAME (see below)
  • -appIcon PATH - Specify an image PATH to display instead of the application icon.
  • -execute COMMAND - Run the shell command COMMAND when the user clicks the notification.
